# 👓 Smart Spectacles for Visually Impaired

## 📌 Project Overview

This project presents an IoT-based Smart Spectacles system designed to assist visually impaired individuals by detecting nearby obstacles and providing real-time alerts through vibration and buzzer feedback.

The system enhances mobility and safety by enabling users to sense obstacles without relying on external assistance.

---

## ⚙️ Features

* 🔍 Obstacle detection using IR sensors
* 🔊 Audio alert using buzzer
* 📳 Vibration feedback for tactile response
* ⚡ Real-time processing using Arduino
* 🔌 Low-cost and portable design

---

## 🛠️ Technologies Used

* Arduino (Microcontroller)
* IR Sensors
* Buzzer
* Vibration Motor
* Embedded C / Arduino IDE

---

## 🔄 How It Works

1. IR sensors continuously scan for nearby obstacles
2. When an object is detected within a threshold distance:

   * Buzzer produces sound alert
   * Vibration motor activates
3. User receives instant feedback and avoids collision
